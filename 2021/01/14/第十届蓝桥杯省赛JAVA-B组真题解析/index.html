import java.io.FileNotFoundException;
import java.util.Scanner;


public class A {
    static int[][] map = new int[20][6];
    static boolean[] flag = new boolean[20];
    static int ans=-1;
    public static void main(String[] args) throws FileNotFoundException {
        Scanner sc = new Scanner(System.in);

        for (int i = 0; i < 20; i++) {
            for (int j = 0; j < 6; j++) {
                map[i][j] = sc.nextInt();
            }
        }
        dfs(1,0);
        System.out.println(ans);
    }

    //cur: 当前所选号位   tmp: 当前总评分
    static void dfs(int cur, int tmp) {
        if (cur == 6) {
            ans = Math.max(ans, tmp);
            return;
        }
        for (int i = 0; i < 20; i++) {
            if (flag[i]) {
                continue;
            }
            flag[i]=true;
            tmp += map[i][cur];
            dfs(cur+1,tmp);
            flag[i]=false;
            tmp -= map[i][cur];
        }
    }
}

B：不同子串

一个字符串的非空子串是指字符串中长度至少为 1 的连续的一段字符组成 的串。例如，字符串aaab 有非空子串a, b, aa, ab, aaa, aab, aaab，一共 7 个。 注意在计算时，只算本质不同的串的个数。

请问，字符串0100110001010001 有多少个不同的非空子串？

分析:
统计非空不重合的总数，很容易想到利用集合去重

答案:

100

import java.util.HashSet;

public class B {
    public static void main(String[] args) {
        String s = "0100110001010001";
        int len = s.length();
        HashSet<String> set = new HashSet<String>();
        for (int i = 0; i < len; i++) {
            for (int j = i+1; j <= len; j++) {
                //左闭右开
                set.add(s.substring(i, j));

            }
        }
        System.out.println(set.size());
    }
}

C：数列求值

给定数列 1, 1, 1, 3, 5, 9, 17, …，从第 4 项开始，每项都是前 3 项的和。求 第 20190324 项的最后 4 位数字。

分析:

类似斐波那契数列,题目要求20190324项的最后四位数字，也就是变相的告诉我们运算过程只和每个数的后四位有关系

答案:

4659

public class C {
    public static void main(String[] args) {
        int a=1,b=1,c=1,tmp=0;
        for (int i = 4; i <=20190324 ; i++) {
            tmp=(a+b+c)%10000;
            a=b;
            b=c;
            c=tmp;
        }
        System.out.println(c);
    }
}

D：数的分解

把 2019 分解成 3 个各不相同的正整数之和，并且要求每个正整数都不包含数字 2 和 4，一共有多少种不同的分解方法？

注意交换 3 个整数的顺序被视为同一种方法，例如 1000+1001+18 和 1001+1000+18 被视为同一种。

分析:

设三个数为i,j,k,且i<j<k
则只要枚举i,j,由k=2019-i-j求得三个数
再确认三个数都不含有2,4即可

答案:

40785

public class D {
    
    //没有2或4返回true
    static boolean check(int n) {
        String s = String.valueOf(n);
        if (s.contains("2") || s.contains("4")) {
            return false;
        }
        return true;
    }

    public static void main(String[] args) {
        int ans = 0;
        int n = 2019;

        for (int i = 1; i < n; i++) {
            if (check(i)) {
                for (int j = i + 1; j < n; j++) {
                    if (check(j)) {
                        int k = n - i - j;
                        if (check(k) && k > j) {
                            ans++;
                        }
                    }
                }
            }
        }
        System.out.println(ans);
    }
}

public class E {
    static int n = 30, m = 50;
    static int[] dx = {1, -1, 0, 0};
    static int[] dy = {0, 0, -1, 1};
    static int[][] dis;
    static char[][] map;
    static StringBuffer sb;
    static String[] dir = {"D", "U", "L", "R"};

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        dis = new int[n][m];
        map = new char[n][m];
        for (int i = 0; i < 30; i++) {
            map[i] = sc.nextLine().toCharArray();
        }
        bfs(n - 1, m - 1);
        dfs();
        System.out.println(sb);
    }

    //标记所有格子到终点的最短路长度
    private static void bfs(int x, int y) {
        Queue<Node> q = new LinkedList<Node>();
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < m; j++) {
                dis[i][j] = -1;
            }
        }
        dis[x][y] = 0;
        q.offer(new Node(x, y));
        while (!q.isEmpty()) {
            Node node = q.poll();
            for (int i = 0; i < 4; i++) {
                int nx = node.x + dx[i];
                int ny = node.y + dy[i];
                if (nx >= 0 && nx < n && ny >= 0 && ny < m && dis[nx][ny] == -1 && map[nx][ny] == '0') {
                    dis[nx][ny] = dis[node.x][node.y] + 1;
                    q.offer(new Node(nx, ny));
                }
            }
        }
    }

    //根据起点到终点的最短路长路按照字典序保存路径
    private static void dfs() {
        int x = 0, y = 0;
        sb = new StringBuffer();
        while (x != n - 1 && y != m - 1) {
            for (int i = 0; i < 4; i++) {
                int nx = x + dx[i];
                int ny = y + dy[i];
                if (nx >= 0 && nx < n && ny >= 0 && ny < m && map[nx][ny] == '0') {
                    if (dis[nx][ny] == dis[x][y] - 1) {
                        x = nx;
                        y = ny;
                        sb.append(dir[i]);
                        break;
                    }
                }
            }
        }
    }
}

class Node {
    int x, y;

    public Node(int x, int y) {
        this.x = x;
        this.y = y;
    }
}

F：特别数的和

小明对数位中含有 2、0、1、9 的数字很感兴趣（不包括前导 0），在 1 到 40 中这样的数包括 1、2、9、10 至 32、39 和 40，共 28 个，他们的和是 574。 请问，在 1 到 n 中，所有这样的数的和是多少？

【输入格式】
输入一行包含一个整数 n。

【输出格式】
输出一行，包含一个整数，表示满足条件的数的和。

【样例输入】
40

【样例输出】
574

分析：

遍历所有数，对每个数判定即可

答案：

public class F {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        long sum = 0;
        for (int i = 1; i <= n; i++) {
            String s = String.valueOf(i);
            if (s.contains("2") || s.contains("0") || s.contains("1") || s.contains("9")) {
                sum += i;
            }
        }
        System.out.println(sum);
    }
}

G：外卖店优先级

“饱了么”外卖系统中维护着 N 家外卖店，编号 1 ∼ N。每家外卖店都有 一个优先级，初始时 (0 时刻) 优先级都为 0。 每经过 1 个时间单位，如果外卖店没有订单，则优先级会减少 1，最低减 到 0；而如果外卖店有订单，则优先级不减反加，每有一单优先级加 2。 如果某家外卖店某时刻优先级大于 5，则会被系统加入优先缓存中；如果 优先级小于等于 3，则会被清除出优先缓存。 给定 T 时刻以内的 M 条订单信息，请你计算 T 时刻时有多少外卖店在优先缓存中。

【输入格式】
第一行包含 3 个整数 N、M 和 T。 以下 M 行每行包含两个整数 ts 和 id，表示 ts 时刻编号 id 的外卖店收到 一个订单。

【输出格式】
输出一个整数代表答案。

【样例输入】
2 6 6
1 1
5 2
3 1
6 2
2 1
6 2

【样例输出】
1

【样例解释】
6 时刻时，1 号店优先级降到 3，被移除出优先缓存；2 号店优先级升到 6， 加入优先缓存。所以是有 1 家店 (2 号) 在优先缓存中。

分析：

用邻接表存储每个时间的订单,这样可以省去排序时间
然后按时刻开始遍历邻接表,更新商店的优先级
最后根据当前商店优先级判定，满足条件进入优先缓存，失去资格踢出优先缓存，缓存使用HashSet，便于处理
输出set元素数即可

答案：

public class G {
    static int n, m, t;
    static int[] shop;//下标编号 值为优先级
    static List<Integer>[] order_t;//下标表示时间 值为该时刻所有订单
    static Set<Integer> set = new HashSet<>();//优先队列

    public static void main(String[] args) throws IOException {
        BufferedReader in = new BufferedReader(new InputStreamReader(System.in));

        String[] date = in.readLine().split("\\s+");
        n = Integer.parseInt(date[0]) + 1;//店数
        m = Integer.parseInt(date[1]);//订单数
        t = Integer.parseInt(date[2]) + 1;//时刻
        shop = new int[n];
        order_t = new List[t];

        for (int i = 0; i < t; i++) {
            order_t[i] = new ArrayList<>();
        }

        for (int i = 0; i < m; i++) {
            date = in.readLine().split("\\s+");
            int ts = Integer.parseInt(date[0]);
            int id = Integer.parseInt(date[1]);
            order_t[ts].add(id);
        }

        //按时间遍历订单记录
        for (int i = 0; i < t; i++) {
            //重置标记
            boolean[] vis = new boolean[n];
            if (!order_t[i].isEmpty()) {
                //如果该时刻有订单,则标记并使优先级增加
                for (int id : order_t[i]) {
                    shop[id] += 2;
                    vis[id] = true;
                }
            }
            //没有订单的店优先级减少
            for (int j = 1; j < n; j++) {
                if (!vis[j] && shop[j] > 0) {
                    shop[j]--;
                }
            }
            //达标进,失格出
            for (int j = 1; j < n; j++) {
                if (shop[j] > 5) {
                    set.add(j);
                } else if (shop[j] <= 3) {
                    set.remove(j);
                }

            }
        }
        PrintWriter out = new PrintWriter(new BufferedOutputStream(System.out));
        out.println(set.size());
        in.close();
        out.close();
    }
}

H：人物相关性分析

小明正在分析一本小说中的人物相关性。他想知道在小说中 Alice 和 Bob 有多少次同时出现。 更准确的说，小明定义 Alice 和 Bob“同时出现”的意思是：在小说文本 中 Alice 和 Bob 之间不超过 K 个字符。 例如以下文本： ThisisastoryaboutAliceandBob.AlicewantstosendaprivatemessagetoBob.
假设 K = 20，则 Alice 和 Bob 同时出现了 2 次，分别是”Alice and Bob” 和”Bob. Alice”。前者 Alice 和 Bob 之间有 5 个字符，后者有 2 个字符。
注意:
1.Alice 和 Bob 是大小写敏感的，alice 或 bob 等并不计算在内。
2.Alice 和 Bob 应为单独的单词，前后可以有标点符号和空格，但是不能有字母。例如 Bobbi 并不算出现了 Bob。

【输入格式】
第一行包含一个整数 K。 第二行包含一行字符串，只包含大小写字母、标点符号和空格。长度不超 过 1000000。

【输出格式】
输出一个整数，表示 Alice 和 Bob 同时出现的次数。

【样例输入】
20
This is a story about Alice and Bob.Alice wants to send a private message to Bob.

【样例输出】
2

分析：

储存所有有效Alice和Bob首字母的位置
获得所有在Alice之前的Bob的数量
获得所有在Bob之前的Alice的数量
相加即为最终结果

答案：

import java.util.ArrayList;
import java.util.List;
import java.util.Scanner;

public class H {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int k = sc.nextInt();
        sc.nextLine();
        String s = sc.nextLine();
        List<Integer> aL = new ArrayList();
        List<Integer> bL = new ArrayList();

        //存储所有Alice
        for (int i = 0; i + 5 < s.length(); i++) {
            //前面不是字母,后面不是字母
            if ((i == 0 || !check(s.charAt(i - 1))) && (i + 5 == s.length() || !check(s.charAt(i + 5)))) {
                if (s.startsWith("Alice", i)) {
                    aL.add(i);
                }
            }
        }
        //存储所有Bob
        for (int i = 0; i + 3 < s.length(); i++) {
            //前面不是字母,后面不是字母
            if ((i == 0 || !check(s.charAt(i - 1))) && (i + 3 == s.length() || !check(s.charAt(i + 3)))) {
                if (s.startsWith("Bob", i)) {
                    bL.add(i);
                }
            }
        }

        long ans = 0;
        //使用-1避免越界
        //Bob--Alice的情况
        for (int i = 0, l = -1, r = -1; i < aL.size(); i++) {
            //最后一个在Alice之前的Bob
            while (r + 1 < bL.size() && aL.get(i) >= bL.get(r + 1)) r++;
            //第一个在Alice之前间隔<=k的Bob
            while (l + 1 <= r && aL.get(i) - (bL.get(l + 1) + 3) > k) l++;
            ans += r - l ;
        }
        //Alice--Bob的情况
        for (int i = 0, l = -1, r = -1; i < bL.size(); i++) {
            while (r + 1 < aL.size() && bL.get(i) >= aL.get(r + 1)) r++;
            while (l + 1 <= r && bL.get(i) - (aL.get(l + 1) + 5) > k) l++;
            ans += r - l ;
        }

        System.out.println(ans);
    }

    static boolean check(char c) {
        return (c >= 'A' && c <= 'Z') || (c >= 'a' && c <= 'z');
    }
}

I：后缀表达式

给定 N 个加号、M 个减号以及 N + M + 1 个整数 A1,A2,··· ,AN+M+1，小 明想知道在所有由这 N 个加号、M 个减号以及 N + M +1 个整数凑出的合法的 后缀表达式中，结果最大的是哪一个？
请你输出这个最大的结果。 例如使用1 2 3 + -，则 “2 3 + 1 -” 这个后缀表达式结果是 4，是最大的。

【输入格式】
第一行包含两个整数 N 和 M。 第二行包含 N + M + 1 个整数 A1,A2,··· ,AN+M+1。

【输出格式】
输出一个整数，代表答案。

【样例输入】
1 1
1 2 3

【样例输出】
4

分析：
后缀表达式就是吓唬人的,平常所用的是中缀表达式,变形就是后缀,后缀只是方便机器运算省略括号而已,这道题按中缀做就好,又不要求输出表达式

因为括号的存在，加号减号之间可以相互转化，当然前提是有减号存在。
当缺少减号可以将加号放在括号中，负正得负， 反之， 负负得正。

所以数据实际上可以自由更改加减，又因为题目要求最大值，则只需要根据以上反转的特性，构造负数，利用一个负数将多个负数都尽可能转换为正值，最终只需要加上最大值减去最小值，再添加完所有的中间值即可求到最大的可能。

其中，当m为0，也就是没有负数时，不需要实施反转操作，所有的操作都为+，此时只需增加所有的数字的绝对值即可。

答案：


import java.util.Scanner;

public class I {

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int m = sc.nextInt();
        int n = sc.nextInt();
        int k = n + m + 1;
        long sum=0;
        int[] arr = new int[200010];
      
        for (int i = 0; i < k; i++) {
            arr[i] = sc.nextInt();
        }

        if (n == 0) {
            for (int i = 0; i < k; i++) {
                sum += arr[i];
            }
        }else {
            int maxp=0,minp=0;
            for (int i = 0; i < k; i++) {
                if (arr[i] > arr[maxp]) {
                    maxp=i;
                }
                if (arr[i] < arr[minp]) {
                    minp=i;
                }
            }

            sum = arr[maxp] - arr[minp];
            
            for (int i = 0; i < k; i++) {
                if (i != maxp && i != minp) {
                    sum += Math.abs(arr[i]);
                }
            }
        }
        System.out.println(sum);
    }
}
